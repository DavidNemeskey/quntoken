# quntoken

New Hungarian tokenizer based on quex, huntoken

## Requirements

* linux
* python 2.x as default python
* python 3.x
* g++ >= 4.8.2
* libicuXX, libicu-dev

## Install

    cd path/to/install/dir/
    git clone https://github.com/dlt-rilmta/quntoken.git
    cd quntoken
    make install
    make all

## Usage

    ./quntoken inputfile
