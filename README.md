# quntoken

New Hungarian tokenizer based on quex and huntoken.

Verison: pre-alpha

## Requirements

* linux
* python 2.x as default python
* python 3.x
* g++ >= 4.8.2
* libicuXX, libicu-dev

## Install

```
cd path/to/install/dir/
git clone https://github.com/dlt-rilmta/quntoken.git
cd quntoken
make install
make all
```

## Usage

```
./quntoken [-f FORMAT] FILE
```
Valid formats: `xml`, `json`  
Default format: `xml`
